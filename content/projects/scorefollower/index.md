## Aim

ScoreFollower is being developed as a desktop editor for synchronizing sheet music with audio and video. The project approaches score following as an editable relationship: musical position and recorded time should be visible together, inspectable, and correctable rather than hidden behind an automatic process.

## Current stage

The public repository marks the beginning of the project. The immediate work is defining the editing model and establishing the application structure before the implementation and documentation are published there.
