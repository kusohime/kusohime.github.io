以滑鼠、觸控筆或觸控操作畫面上的琴鍵；各個獨立指標可以同時按住多個音。
琴鍵上印出的 QWERTY 對照會把電腦鍵盤變成同一組音高。對照採用實體按鍵位置
（`KeyboardEvent.code`），因此即使不同鍵盤配置在按鍵上印出不同字符，演奏的
手形仍保持一致。

若瀏覽器提供 Web MIDI，可在授予權限後使用已連接的輸入裝置，以硬體控制器
演奏同一套合成器。MIDI 傳送的是音符與控制事件，而非聲音本身；因此鍵下與鍵起
訊息會開啟及釋放本機合成聲部，力度決定音量，延音踏板則延續已放開的音。指標、
QWERTY 與 MIDI 輸入可以混合使用；緊急停止控制會釋放所有仍在發聲的聲部。

所有聲音都以 Web Audio API 在本機瀏覽器內合成。這些音色刻意保持基本，並能
乾淨地持續發聲，讓音程、音域配置與聲部進行可以按需要聽得足夠久。它們是聆聽
輔助，而非對真實樂器的取樣模仿；切換音色改變的是頻譜與包絡，不會改變正在
按住的音高。

本工具不要求麥克風權限、不傳送 MIDI 輸出，也不會上傳或保存音符、控制訊息或
錄音。MIDI 存取由瀏覽器的權限系統處理，普通輸入存取不要求系統專屬訊息
（SysEx）。列舉裝置本身可能透露可識別身分的硬體資訊，因此只有在使用者明確
提出要求後才會開始存取。

Web MIDI 是可選功能：它需要瀏覽器支援、安全來源、使用者權限及已連接的裝置。
其中任何一項缺少時，指標與 QWERTY 鍵盤仍可使用。QWERTY 輸入沒有獨立力度或
觸後，觸控行為與聲音延遲會因裝置而異，瀏覽器亦可能在第一次使用者操作之前
暫停音訊。本工具不是音序器、錄音機、General MIDI 音源、標準 MIDI 檔案播放
器或 MIDI 輸出工具；它是一件用於試聽和聲的精簡多聲部樂器。
