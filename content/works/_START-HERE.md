# Works / 作品

**中文：** 每个作品使用一个文件夹，文件夹内的 `index.md` 是作品资料与说明。
新增作品时，复制一个现有作品文件夹并修改即可。

**English:** Each work has one folder. Its `index.md` contains metadata and
program notes. To add a work, copy an existing work folder and edit the copy.
