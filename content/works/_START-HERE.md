# Works / 作品

**中文：** 每個作品使用一個資料夾，資料夾內的 `index.md` 是作品資料與說明。
新增作品時，複製一個現有作品資料夾並修改即可。

**English:** Each work has one folder. Its `index.md` contains metadata and
program notes. To add a work, copy an existing work folder and edit the copy.

See `content/README.md` for all fields and examples.

全部欄位與示例請看 `content/README.md`。
