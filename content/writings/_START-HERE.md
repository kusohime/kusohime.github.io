# Writings / 文章

**中文：** 每篇文章使用一個資料夾。`index.md` 是主檔案；只有分章文章才需要
`chapters/` 資料夾。網站會自動根據章節檔案生成目錄。

**English:** Each writing has one folder. `index.md` is the main file; add a
`chapters/` folder only for a multi-part writing. The website generates the
contents list automatically.

See `content/README.md` for step-by-step examples.

逐步示例請看 `content/README.md`。
