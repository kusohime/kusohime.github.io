### <span class="libretto-source" lang="it" id="room-xv">Stanza XV – Remigio &lt;Sol&gt;</span><br /><span class="libretto-translation" lang="en">Room XV – Remigio &lt;G&gt;</span>

**Remigio**

**Guglielmo**

<span class="libretto-source" lang="it">Nel villaggio a valle</span><br />

**Remigio**  
<span class="libretto-source" lang="it">Sì e no,</span><br />

<span class="libretto-source" lang="it">coi quali dividiamo</span><br />

<span class="libretto-source" lang="it">alcuni nostri beni.</span><br />

<span class="libretto-source" lang="it">Polli, giovenche,</span><br />
<span class="libretto-translation" lang="en">Chickens, heifers,</span><br />
<span class="libretto-source" lang="it">vitelli, buoi, cavalli,</span><br />
<span class="libretto-translation" lang="en">calves, oxen, horses,</span><br />
<span class="libretto-source" lang="it">maiali affumicati,</span><br />
<span class="libretto-translation" lang="en">smoked pigs,</span><br />
<span class="libretto-source" lang="it">pecore, alveari,</span><br />
<span class="libretto-translation" lang="en">sheep, beehives,</span><br />
<span class="libretto-source" lang="it">miele, malto, strutto,</span><br />
<span class="libretto-translation" lang="en">honey, malt, lard,</span><br />
<span class="libretto-source" lang="it">sapone…</span><br />
<span class="libretto-translation" lang="en">soap…</span>

**Guglielmo**  
<span class="libretto-source" lang="it">Va bene, va bene,</span><br />
<span class="libretto-translation" lang="en">All right, all right,</span><br />
<span class="libretto-source" lang="it">e quanta terra ha</span><br />
<span class="libretto-translation" lang="en">and how much land does</span><br />
<span class="libretto-source" lang="it">chi non ha questa fortuna?</span><br />
<span class="libretto-translation" lang="en">one who lacks this fortune have?</span>

**Remigio**  
<span class="libretto-source" lang="it">Cinquanta tavole a famiglia.</span><br />
<span class="libretto-translation" lang="en">Fifty tavole per family.</span>

**Guglielmo**  
<span class="libretto-source" lang="it">E una tavola quant’è?</span><br />
<span class="libretto-translation" lang="en">And how much is one tavola?</span>

**Remigio**

<span class="libretto-translation" lang="en">Four square trabucchi.</span>

**Guglielmo**  
<span class="libretto-source" lang="it">E i trabucchi quanto sono?</span><br />
<span class="libretto-translation" lang="en">And how much are the trabucchi?</span>

**Remigio**  
<span class="libretto-source" lang="it">Un trabucco quadro</span><br />
<span class="libretto-translation" lang="en">One square trabucco</span><br />
<span class="libretto-source" lang="it">fa trentasei piedi.</span><br />

<span class="libretto-source" lang="it">Ottocento lineari</span><br />
<span class="libretto-translation" lang="en">Eight hundred linear ones</span><br />
<span class="libretto-source" lang="it">fanno un miglio piemontese.</span><br />

<span class="libretto-source" lang="it">Per ogni famiglia conta</span><br />
<span class="libretto-translation" lang="en">For each family count</span><br />

<span class="libretto-translation" lang="en">half a sack of oil.</span>

**Guglielmo**  
<span class="libretto-source" lang="it">Mezzo sacco d’olio?</span><br />
<span class="libretto-translation" lang="en">Half a sack of oil?</span>

**Remigio**  
<span class="libretto-source" lang="it">Fa cinque emine,</span><br />
<span class="libretto-translation" lang="en">That makes five emine,</span><br />
<span class="libretto-source" lang="it">un’emina otto coppe.</span><br />
<span class="libretto-translation" lang="en">one emina eight cups.</span>

**Guglielmo**  
<span class="libretto-source" lang="it">E in boccali?</span><br />
<span class="libretto-translation" lang="en">And in tankards?</span>

<span class="libretto-source" lang="it">Rubbie…</span><br />
<span class="libretto-translation" lang="en">Rubbie…</span>

**Remigio**  
<span class="libretto-source" lang="it">Sei rubbie una brenta.</span><br />
<span class="libretto-translation" lang="en">Six rubbie to one brenta.</span>

**Guglielmo**  
<span class="libretto-source" lang="it">Brente…</span><br />
<span class="libretto-translation" lang="en">Brente…</span>

**Remigio**  
<span class="libretto-source" lang="it">Otto brente un bottale.</span><br />
<span class="libretto-translation" lang="en">Eight brente to one bottale.</span>

**Guglielmo**  
<span class="libretto-source" lang="it">Bottali…</span><br />
<span class="libretto-translation" lang="en">Bottali…</span>

**Remigio**

**Guglielmo**

<span class="libretto-source" lang="it">che non abbia alcun trabucco</span><br />

**Remigio**  
<span class="libretto-source" lang="it">Un trabucco… trentasei piedi…</span><br />
<span class="libretto-translation" lang="en">One trabucco… thirty-six feet…</span>

**Guglielmo**
